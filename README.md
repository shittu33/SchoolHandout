## App Features
* Signup
* Login
* Profile
* Local Handouts
* Online Hanouts
* Curl Page Flip
* Book Discussion

## Handout Features
* Flip Book
* Pdf to Handout
* Camera to Handout
* Gallery to Handout
* Covert to Pdf

## Backend
* Firebase
* Sqlite
